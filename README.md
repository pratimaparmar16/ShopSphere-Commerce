# ShopSphere-Commerce

ShopSphere Commerce is a production-style e-commerce platform created to demonstrate enterprise-level software development using C#,  ASP.NET Core MVC (.NET 10), Entity Framework Core, SQL Server, Clean Architecture, REST APIs, CCAvenue Payment Gateway Integration, Angular, and React.

The project is being developed step-by-step from scratch with a focus on software engineering principles, scalability, maintainability, and real-world business requirements.
